# Наші роботи (галерея)

Папка: `public/images/gallery/`

## Як називати фото

| Файл | Опис |
|------|------|
| `1.webp` | Перше фото |
| `2.webp` | Друге |
| `3.webp` | Третє |
| … | Далі по порядку |

**Формат:** WEBP (або JPG — напишіть, підлаштуємо)  
**Розмір:** орієнтовно **1200×900 px**, до **500 КБ** на файл  
**Співвідношення:** 4:3 або 16:9 — однакове виглядає краще в сітці

## Після додавання

1. Покладіть файли сюди (`1.webp`, `2.webp`, …).
2. Напишіть скільки фото — підключимо на сайті в `lib/gallery.ts`.

Оновлення сторінки: **Cmd+Shift+R**.
