# Hero-фото для сторінок послуг і міст

Для кожної сторінки — **два** файли (додати один раз):

| Файл | Орієнтація | Приклад |
|------|------------|---------|
| `{slug}-desktop.webp` | горизонтальне | `boyarka-desktop.webp` |
| `{slug}-mobile.webp` | вертикальне | `boyarka-mobile.webp` |

**slug** — як у URL: `boyarka`, `vyviz-budivelnogo-smittya` (seoPath послуги).

Поки файлів немає — показується фото з галереї (fallback).

Рекомендований розмір: desktop ~1920×1080, mobile ~1080×1920, WEBP до 500 КБ.
