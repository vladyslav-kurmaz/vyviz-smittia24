# Головний екран (Hero)

## Фон головної сторінки

| Файл | Роль |
|------|------|
| **`hero.mp4`** | Фонове відео **десктоп** (горизонтальне, зациклене, без звуку) |
| **`hero-mobile.mov`** | Фонове відео **мобайл** (вертикальне 9:16, зациклене, без звуку) |
| `hero-bg.jpg` | Запасне статичне фото, якщо відео не завантажиться |

Статичні постери: десктоп — `gallery/1.webp`, мобайл — `gallery/7.webp` (вертикальне).

На сайті гіфка показується **на весь фон** за текстом, як раніше фото — не окремою карткою.

Якщо заміните GIF — перегенеруйте MP4 (FFmpeg або ezgif.com), інакше залишиться старий ролик.

**Cmd+Shift+R** після заміни файлів.
