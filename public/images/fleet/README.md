# Фото автопарку (калькулятор)

Покладіть сюди фото машин для блоку «Ціни на подачу авто».

| Ім’я файлу | Машина |
|------------|--------|
| `fleet-bus.webp` | Бус |
| `fleet-gazel.webp` | Газель |
| `fleet-zil.webp` | ЗІЛ |
| `fleet-kamaz.webp` | Камаз |
| `tractor.webp` | Трактор |

**Формат:** WEBP або JPG  
**Розмір:** орієнтовно **800×500** px (горизонтально, машина добре видна)  
**Вага:** до **500 КБ** на файл (щоб швидко вантажилось)

Після додавання — `Cmd+Shift+R` на сайті.

Шлях у проєкті:

```
vyviz-landing/public/images/fleet/
```

Імена та ціни змінюються в `lib/fleet.ts`.
