# Логотипи месенджерів

Покладіть сюди **3 файли** — білий логотип на **прозорому** фоні (без кольорового круга).

| Ім’я файлу | Сервіс |
|------------|--------|
| `messenger-telegram-white.svg` | Telegram |
| `messenger-viber-white.svg` | Viber |
| `messenger-whatsapp-white.svg` | WhatsApp |

**Формат:** SVG (краще) або PNG  
**Розмір:** 24×24 або 32×32 px  
**Колір:** білий `#FFFFFF` (круглий фон малює сайт — зелений або напівпрозорий у хедері)

Після додавання — `Cmd+Shift+R` на сторінці.

Якщо файлу немає, показуються вбудовані SVG-запасні іконки.

Імена змінюються в `lib/messengers-ui.ts`.
