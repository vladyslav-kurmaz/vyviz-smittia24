# Іконки блоку «Переваги роботи з нами»

Покладіть сюди **5 файлів** (SVG краще за PNG; розмір **48×48** або **64×64** px).

| Файл | Про що іконка |
|------|----------------|
| `benefit-fleet.svg` | Власний автопарк (вантажівка / автопарк) |
| `benefit-fast.svg` | Швидка подача авто (~30 хв) |
| `benefit-experience.svg` | Досвід / роки на ринку |
| `benefit-legal.svg` | Легальна утилізація / документи / екологія |
| `benefit-247.svg` | Робота 24/7 (годинник, цілодобово) |

Після додавання файлів перезавантажте сторінку (`Cmd+Shift+R`).  
Поки файлу немає — на сайті показується emoji-запасний варіант.

Імена змінюються в `lib/benefits.ts` (поле `iconFile`).
