# Карта зони роботи

Покладіть сюди зображення карти для блоку **«Де ми працюємо»** на головній.

| Параметр | Значення |
|----------|----------|
| **Ім'я файлу** | `service-area-map.webp` |
| **Шлях** | `public/images/geography/service-area-map.webp` |
| **Формат** | WEBP або JPG |
| **Розмір** | орієнтовно **1200×800 px** (горизонтально) |
| **Вага** | до **500 КБ** |

Можна скріншот Google Maps / OpenStreetMap з виділеною зоною Києва та області, або дизайнерську карту з вашим брендингом.

Після додавання — оновіть сторінку (**Cmd+Shift+R**).
