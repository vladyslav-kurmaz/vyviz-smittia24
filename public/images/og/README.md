# Open Graph — зображення для соцмереж і месенджерів

## Обовʼязково: одне головне зображення

| Параметр | Значення |
|----------|----------|
| **Файл** | `og-default.webp` |
| **Шлях** | `public/images/og/og-default.webp` |
| **Розмір** | **1200 × 630 px** (співвідношення 1.91:1) |
| **Формат** | WEBP або JPG |
| **Вага** | до **300 КБ** |

### Що має бути на картинці

1. **Фон** — зелений градієнт бренду (`#233d28` → `#355e3b`) або фото вантажівки/робіт з напівпрозорим затемненням.
2. **Логотип / назва** — «ВИВІЗ СМІТТЯ **24**» (24 — помаранчевим `#D97708`).
3. **Підзаголовок** — «Київ та область» або «Швидкий вивіз сміття».
4. **Опційно** — фото буса/газелі з галереї праворуч (не перекривати текст).
5. **Опційно** — телефон `096 400 41 41` дрібнішим шрифтом внизу.

**Безпечна зона:** важливий текст і логотип — у центральних 80% кадру (краї обрізаються в Telegram / Facebook).

Це зображення використовується на **усіх сторінках**, поки не додасте окремі (нижче).

## Опційно: окремі OG для міст і послуг

Якщо потрібні унікальні превʼю в посиланнях:

| Тип | Імʼя файлу | Приклад тексту на зображенні |
|-----|------------|------------------------------|
| Місто | `og-city-boyarka.webp` | «Вивіз сміття Боярка» |
| Послуга | `og-service-budivelne.webp` | «Вивіз будівельного сміття» |

Розмір той самий: **1200 × 630 px**. Поки файлів немає — скрізь показується `og-default.webp`.

Після додавання — напишіть, підключимо в коді для конкретних сторінок.
